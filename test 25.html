HTML Basics

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It consists of elements, which are the building blocks of web pages.

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. It can be used to change colors, fonts, layouts, and more.

body {
    background-color: lightblue;
}

h1 {
    color: navy;
    margin-left: 20px;
}
